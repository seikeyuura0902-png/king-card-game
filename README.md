# ♛ キングカードゲーム

9枚のカードで戦略バトル！AI対戦・オンライン対戦対応

## 🚀 起動方法

### 必要なもの
- Node.js (v16以上) https://nodejs.org/

### 手順

```bash
# 1. ZIPを解凍（またはフォルダに移動）
cd king-card-game

# 2. 依存関係インストール（初回のみ）
npm install

# 3. サーバー起動
npm start

# 4. ブラウザで開く
# → http://localhost:3000
```

### スマートフォンでのアクセス（同一Wi-Fi内）
1. サーバーを起動したPCのIPアドレスを確認
   - Windows: `ipconfig` コマンド
   - Mac/Linux: `ifconfig` コマンド
2. スマートフォンのブラウザで `http://[PCのIPアドレス]:3000` にアクセス

## 🎮 ゲームルール概要

### 勝利条件
相手のカードを**6枚**倒す

### カード一覧
| カード | 勝てる相手 |
|-------|-----------|
| 👑 貴族 | 奴隷・将軍（兵士は相打ち） |
| ⚔️ 将軍 | 奴隷・兵士 |
| 🛡️ 兵士 | 奴隷・市民（貴族は相打ち） |
| 🏘️ 市民 | 奴隷・貴族・将軍 |
| ⛓️ 奴隷 | 特殊カード全て |
| 🏯 皇帝（特殊） | 通常カード・始皇帝 |
| 🐉 始皇帝（特殊） | 通常カード・狙撃手・革命家 |
| 🎯 狙撃手（特殊） | 通常カード・皇帝 |
| 🔥 革命家（特殊） | 通常カード・皇帝 |

### 特殊能力（自分のカードが2枚死んだら解禁）
- **皇帝「勅命」**: A: 相手の手札1枚を3ターン禁止 / B: 次ターン相手は兵士か奴隷のみ
- **始皇帝「万里の長城」**: 3ターン間負けない（負け→引分、相打ち→自分生存）
- **狙撃手「暗殺」**: 相手の手札1枚を殺す（奴隷不可・勝利条件外）
- **革命家「革命」**: 自分の死んだカード2枚を復活（3枚死亡後のみ・復活分は勝利条件外）

## 🛠️ 技術仕様

- **フロントエンド**: HTML5 / CSS3 / Vanilla JavaScript
- **バックエンド**: Node.js + Express + Socket.io
- **通信最適化**: Client-Side Prediction（CSP）+ Interpolation
- **対応ブラウザ**: Chrome / Safari / Firefox / Edge（モバイル対応）
